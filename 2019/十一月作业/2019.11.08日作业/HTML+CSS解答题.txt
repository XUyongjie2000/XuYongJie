1.css的三大特性分别是什么，请详细说明，并且举例子
层叠性:相同选择器中的相同属性 哪个选择器离标签近 就用哪个 就近原则 层叠也叫覆盖
继承性: 子承父业  例如 font-text-line-color
优先级:不同得选择器作用于同一个元素上，权重最高的那个被优先使用
2.css哪些属性能够继承，举例不得少于5个
font text line color  list
3.css权重计算规则
内联样式：权值为1000
id选择器：权值为0100
类选择器：权值为0010
标签和伪元素选择器：权值为0001
通配符 子选择器 相邻选择器：权值为0000
4.浮动的元素有什么特点
浮动元素会脱离正常的文档流，按照其外边距指定的位置相对于它的上一个块级元素（或父元素）显示
浮动元素后面的块级元素的内容会向此浮动元素的外边距靠齐，但是边框和背景却忽略浮动元素而向上一个任意非浮动元素靠齐
浮动元素后面的内联元素会向此浮动元素的外边距靠齐
